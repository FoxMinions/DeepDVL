## 项目简介

**DeepDVL**（深度数界实验室）是一个以「教学」为唯一目的的开源项目。它不会一次性丢给你一个完成度很高的大屏成品，而是从一张空页面开始，一步步带你理解数据可视化大屏是如何被「造」出来的——从布局骨架、适配方案、图表选型，到数据流转、交互联动、动效打磨。

我们的目标不是让你「有一个能跑的 demo」，而是让你**真正理解每一行代码为什么这么写**，最终能脱离教程，独立做出属于自己的大屏。

## 适合谁

- 前端开发者，想做数据可视化但不知道从何下手
- 有一定基础，想系统学习大屏的**布局适配、图表封装、工程化**实践
- 想要一个可以边看边动手、能持续跟着迭代的学习项目

## 能学到什么

> 📌 以下为项目规划，随内容迭代持续更新。

- 数据大屏的布局与自适应方案（rem / scale / flexible 等思路对比）
- 常见图表库的封装与使用（ECharts / 其他可视化方案）
- 数据的模拟、请求、流转与状态管理
- 组件化开发与复用：标题栏、边框、装饰、地图、排行等通用组件
- 交互联动、定时刷新、动效与细节打磨
- 工程化与代码组织：目录结构、规范、可维护性
