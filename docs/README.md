# DeepDVL 文档中心

这里存放 **DeepDVL · 深度数界实验室** 的全部教学文档与分阶段提示词。

## 这是什么

DeepDVL 是一个公开开源的实战项目，目标是带你 **从 0 到 1 亲手做出一个数据可视化大屏**。

和「一次性给你一个成品」不同，本项目把大屏的开发过程拆成 **9 个可独立验证的阶段**，每个阶段：

- 只做一件事，产出可运行、可验证的代码；
- 配一份 **小提示词**，你可以直接喂给 AI（Claude / Cursor / Copilot 等）让它帮你实现；
- 配一段 **「为什么这么做」** 的讲解，讲清设计理由，而不是只让你照抄；
- 给一份 **验收清单**，做完自己勾选确认。

## 怎么用

1. 先读 [`提示词/00-项目总览与使用说明.md`](./提示词/00-项目总览与使用说明.md) —— 了解技术栈、视觉基线、9 阶段全貌。
2. 按顺序进入每个阶段文档，把里面 **「执行提示词」** 一节的内容复制给 AI。
3. AI 实现完成后，按 **「验收清单」** 逐项检查。
4. 通过后 `git commit`，进入下一阶段。

> 阶段之间有依赖关系，建议**按顺序做**，不要跳。每份阶段文档开头都写了「前置依赖」。
